# wordpretty

![wordpretty](./docs/wordpretty.png)


## Description

コメント中の文字列をイラストに置き換えるわんコメプラグイン

兼、わんコメでコメントを置き換えるようなプラグインの開発プロジェクトサンプル


## Requirements

- Bun v1.1.29+


## Development

- vite: commonjs形式へのビルド
- biome: コード整形
- bun-test: 単体テスト
- github-actions: 自動テスト、自動リリース
